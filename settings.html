Bitcoin Address Inspector

Configure how Bitcoin addresses are detected and displayed

Detection Methods

Show address information when hovering over detected Bitcoin addresses

Show context menu option when right-clicking on selected Bitcoin addresses

Hover Settings

500ms

How long to wait before showing the popup when hovering

Information Display

Display the current Bitcoin balance of the address

Display the number of unspent transaction outputs

Display the total amount of Bitcoin ever received by this address

Data Source

API Source: mempool.space

This extension uses the free mempool.space API to fetch Bitcoin address information. No API key required.
